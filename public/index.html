Secure External Reference Number Exchange

Welcome to the Reference Exchange Portal

This portal facilitates secure exchange of reference numbers between suppliers and customers. Choose your role below to get started.

For Suppliers

Submit and manage reference numbers for purchase orders and deliveries.

  • Submit reference numbers individually or via CSV upload
  • View submission history
  • Secure email-based authentication
  • Update existing submissions
Access Supplier Portal

For Customers

Access reference numbers for your purchase orders and deliveries.

  • Access via postcode or email verification
  • Download references as CSV
  • Generate shareable links
  • Mobile-friendly interface
Access Customer Portal

How it Works

Suppliers

  1. Access via unique supplier link
  2. Verify email with OTP
  3. Submit reference numbers
  4. Track submission history

Customers

  1. Enter PO/Delivery number
  2. Verify with postcode or email
  3. View reference numbers
  4. Download or share as needed
